#  Java Course Repository

This repository is a structured collection of my **Java learning journey**, organized around two main textbooks and their related exercises. Each book is divided into **course materials** (concepts, notes, and chapter outlines) and **exercise folders** (practical coding tasks and problem solutions).

---

## 📘 Repository Structure

```
Java_Course/
├── An_Interdisciplinary_Approach/
│   ├── course/         → Notes, summaries, and examples from each chapter
│   └── exercise/       → Coding exercises and practice files
│
├── Building_Java_Program/
│   ├── course/         → Course notes, chapter breakdowns, and code snippets
│   └── exercise/       → Exercises and small projects by chapter
│
├── java_vscode_setup.md → Setup guide for Java and VSCode environment
└── README.md            → This file
```
