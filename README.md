# Java Programming Course

This repository contains my personal guide and practice work for the **Java Programming** course.  

I will be studying each week’s material using the following books:

- *Building Java Programs: A Back to Basics Approach (5th Edition)*  
  by **Stuart Reges** and **Marty Stepp**  

- *Computer Science: An Interdisciplinary Approach*  
  by **Robert Sedgewick** and **Kevin Wayne**  

## Repository Structure
- **courses/** → Contains notes organized by topic and weekly materials.  
- **exercises/** → Contains practice files and coding exercises.  

## Purpose
- Build a long-term **reference guide** for Java programming concepts.  
- Reinforce understanding through **weekly practice**.  
- Maintain a clear separation between **theory (courses)** and **practice (exercises)**.
